# kanberoo-web

**Placeholder.** This package name is reserved for the Kanberoo phase 2 web UI. Not yet released.

Kanberoo phase 1 ships a Textual TUI, a Typer CLI, a FastAPI REST + WebSocket server, and an MCP server. Phase 2 adds a web UI; the stack was deliberately deferred so the final TUI could inform the design. See [`docs/spec.md` section 9.2](https://github.com/areese801/kanberoo/blob/main/docs/spec.md).

For everything that IS shipping today:

```bash
pip install 'kanberoo[all]'
```
