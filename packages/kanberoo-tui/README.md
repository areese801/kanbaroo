# kanberoo-tui

Textual terminal UI for [Kanberoo](https://github.com/areese801/kanberoo), a kanban-style issue tracker. Workspace list, kanban board, story detail with markdown, comments, linkages, tags, and an audit feed. Live updates via WebSocket.

```bash
pip install kanberoo-tui
kanberoo-tui
```

Requires a running `kanberoo-api` server and a config initialized via `kb init` (from `kanberoo-cli`).

Full docs: [github.com/areese801/kanberoo](https://github.com/areese801/kanberoo).
