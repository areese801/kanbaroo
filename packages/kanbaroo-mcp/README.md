# kanbaroo-mcp

MCP server for [Kanbaroo](https://github.com/areese801/kanbaroo), a kanban-style issue tracker. Exposes Kanbaroo's workspaces, epics, stories, comments, tags, linkages, and audit trail as MCP tools so an AI agent can read and write the board.

```bash
pip install kanbaroo-mcp
```

Setup guide for Claude Desktop / Claude Code: [docs/mcp-setup.md](https://github.com/areese801/kanbaroo/blob/main/docs/mcp-setup.md).
