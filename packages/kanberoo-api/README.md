# kanberoo-api

FastAPI REST + WebSocket server for [Kanberoo](https://github.com/areese801/kanberoo), a kanban-style issue tracker with a TUI, REST + WebSocket API, CLI, and MCP server.

```bash
pip install kanberoo-api
kanberoo-api   # requires KANBEROO_DATABASE_URL
```

Full docs and configuration: [github.com/areese801/kanberoo](https://github.com/areese801/kanberoo).
