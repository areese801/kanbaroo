# kanberoo-core

Shared SQLAlchemy models, Pydantic schemas, services, audit emission, WebSocket event bus, and Alembic migrations for [Kanberoo](https://github.com/areese801/kanberoo), a kanban-style issue tracker with a TUI, REST + WebSocket API, CLI, and MCP server.

This package is a runtime dependency of every other Kanberoo package. To install everything at once:

```bash
pip install 'kanberoo[all]'
```

Full docs: [github.com/areese801/kanberoo](https://github.com/areese801/kanberoo).
