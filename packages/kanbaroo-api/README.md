# kanbaroo-api

FastAPI REST + WebSocket server for [Kanbaroo](https://github.com/areese801/kanbaroo), a kanban-style issue tracker with a TUI, REST + WebSocket API, CLI, and MCP server.

```bash
pip install kanbaroo-api
kanbaroo-api   # requires KANBAROO_DATABASE_URL
```

Full docs and configuration: [github.com/areese801/kanbaroo](https://github.com/areese801/kanbaroo).
